<h2> How to Run: </h2>

- Clone this project
- Navigate to the project directory
  
<h3>Terminal></h3>

```Set-ExecutionPolicy Restricted -Scope CurrentUser``` ❌

```Set-ExecutionPolicy RemoteSigned -Scope CurrentUser``` ✅

``` npm install -g yarn  ```

``` yarn  install```

``` yarn  run dev```


<h2>Getting Started: How to Create React + Vite Project</h2>

<h3>Terminal></h3>

``` yarn  create vite```
